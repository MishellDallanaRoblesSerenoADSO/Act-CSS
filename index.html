Selectores

Tipos de selectores:

  • * : Selecciona todos los elementos
  • De tipo o etiqueta : Selecciona por el nombre de los elementos
  • Clases : Se le asigna una clase a cada elemento y se selecciona con un punto: .h3{}
  • Id : Se usa para identificar un elemento que no se repite, es decir un elemento unico y se selecciona con el #: #(nombre del id){}
  • Por atributo: composicion etiqueta HTML Nombre de etiqueta, atributo = "valor del atributo". El atributo se selecciona con []: [hola="21"]{}

    • Descendiente:

    Se selecciona primero al contenedor, luego al descendiente: h2 p

  • Pseudo clases: Son los eventos de la seleccion.
